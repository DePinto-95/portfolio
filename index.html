BSc Thesis

An Interactive Minesweeper Solver Using an SMT Solver
Vrije Universiteit Amsterdam · 2024

Built a terminal-based solver in Scala that encodes the Minesweeper board as integer constraints and uses the Z3 SMT solver to deduce — with mathematical certainty — which cells are safe and which are mines.

MSc Thesis

Hybrid Android App Debloating
Chalmers University of Technology · In progress

Android apps often ship with far more code and permissions than they actually need — making them larger, slower, and less private. My thesis builds a tool that automatically strips out the unused parts: first removing permissions the app declares but never uses, then removing code paths that real users never exercise.
